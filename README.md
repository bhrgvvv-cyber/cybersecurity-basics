## CIA Triad

- **Confidentiality** → Information must be kept secret
- **Integrity** → Information should not be changed without permission
- **Availability** → Information should be accessible when needed

## Types of Hackers

- White Hat → Ethical hackers
- Black Hat → Malicious hackers
- Grey Hat → Between ethical and malicious
